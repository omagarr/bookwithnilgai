Ski-Lifts Chat Widget Example

This page demonstrates the floating widget version of the Ski-Lifts chat interface.

About the Widget

The widget version creates a floating chat interface that:

  • Appears as a button in the corner of the page
  • Can be minimized and maximized
  • Maintains state across page sessions
  • Works on all screen sizes

Integration Code

Add this simple script tag to your website:

<script src="https://ski-lifts.nilgai.travel/widget.js"></script>
