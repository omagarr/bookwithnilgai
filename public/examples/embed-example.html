Ski-Lifts Chat Embed Examples

Inline embeddable chat interface that adapts to any container

About the Embed Version: Unlike the floating widget, the embed version is designed to be placed inline within your page content. It automatically adapts to the size of its container and is always in expanded form.

📊 Features

  • Inline placement within any container
  • Responsive and adapts to container size
  • No minimize/maximize - always expanded
  • Perfect for banners, sidebars, or dedicated sections
  • Multiple embeds on the same page
  • Customizable dimensions and styling

🚀 Quick Start

Add the embed with just two lines of code:

<div data-ski-lifts-embed></div> <script src="https://ski-lifts.nilgai.travel/embed.js"></script>

Or programmatically:

<div id="chat-container"></div> <script src="https://ski-lifts.nilgai.travel/embed.js"></script> <script> SkiLiftsEmbed.init('#chat-container', { height: '500px', width: '100%' }); </script>

🎨 Banner Embed Example

Perfect for hero sections or promotional banners:

<div class="banner-container" data-ski-lifts-embed data-embed-height="100%"></div>

📱 Sidebar Embed Example

Great for sidebars or dedicated help sections:

<div id="sidebar-embed" style="height: 600px;"></div> <script> SkiLiftsEmbed.init('#sidebar-embed', { height: '100%', borderRadius: '0' }); </script>

⚙️ Advanced Configuration

Use data attributes for configuration:

<div data-ski-lifts-embed data-embed-height="500px" data-embed-width="100%" data-embed-border-radius="12px" data-embed-class="custom-chat-class" ></div> <script src="https://ski-lifts.nilgai.travel/embed.js"></script>

Initialize with JavaScript for more control:

<script src="https://ski-lifts.nilgai.travel/embed.js"></script> <script> // Wait for DOM ready SkiLiftsEmbed.ready.then(() => { // Initialize embed with custom options const iframe = SkiLiftsEmbed.init('#my-container', { height: '600px', width: '100%', borderRadius: '16px', style: { boxShadow: '0 10px 30px rgba(0,0,0,0.2)' } }); // You can access the iframe element if needed console.log('Embed initialized:', iframe); }); </script>

You can have multiple embeds on the same page:

<div id="embed1"></div> <div id="embed2"></div> <script src="https://ski-lifts.nilgai.travel/embed.js"></script> <script> // Initialize multiple embeds SkiLiftsEmbed.init('#embed1', { height: '400px' }); SkiLiftsEmbed.init('#embed2', { height: '500px' }); // Or auto-initialize all with data attributes SkiLiftsEmbed.initAll(); </script>
