E-commerce Sites

Use the widget for subtle support presence without interrupting the shopping experience.

Widget Recommended

Embed the chat directly in your contact section as the primary communication method.

Embed Recommended

Landing Pages

Feature the embed prominently to encourage immediate engagement with visitors.

Embed Recommended

Documentation Sites

Float the widget to provide contextual help without covering important content.

Widget Recommended

Support Centers

Embed as the main interface in dedicated support sections or help desks.

Embed Recommended

Corporate Websites

Use both: widget for general pages, embed for contact/support sections.

Integration Code

Simple Widget Integration

<!-- Add this before closing </body> tag --> <script src="https://ski-lifts.nilgai.travel/widget.js"></script> <!-- Optional: Custom configuration --> <script> window.__CHAT_WIDGET_CONFIG = { position: 'bottom-right', initiallyMinimized: true }; </script>

Simple Embed Integration

<!-- Add where you want the chat to appear --> <div data-ski-lifts-embed style="height: 500px;"></div> <script src="https://ski-lifts.nilgai.travel/embed.js"></script> <!-- Or with custom options --> <div id="chat-container" style="height: 600px;"></div> <script src="https://ski-lifts.nilgai.travel/embed.js"></script> <script> SkiLiftsEmbed.init('#chat-container', { height: '100%', borderRadius: '12px' }); </script>

Using Both Widget and Embed

<!-- Embed in your contact section --> <section id="contact"> <h2>Get in Touch</h2> <div data-ski-lifts-embed style="height: 500px;"></div> </section> <!-- Scripts at the bottom of the page --> <script src="https://ski-lifts.nilgai.travel/embed.js"></script> <script src="https://ski-lifts.nilgai.travel/widget.js"></script> <!-- Both will work independently without conflicts -->
